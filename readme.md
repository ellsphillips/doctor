# Doctor

An automated documentation assitant for Pythonic data markup to LaTeX.

## Installation

```shell
  doctor/
  ├── data
  │   ├── __init__.py
  │   ├── data.py
  ├── demo
  │   ├── src
  │   │   └── ...
  │   ├── doctor.cls
  │   ├── main.tex
  │   ├── report.pdf
  ├── doctor
  │   ├── builder
  │   │   ├── plot
  │   │   └── table
  │   ├── utils
  │   │   ├── __init__.py
  │   │   ├── cli.py
  │   │   └── tex.py
  │   ├── __init__.py
  │   ├── doctor.py
  └── app.py
```
